
NAPALM Ex1
-----------
1. Construct a script that retrieves facts from two IOS routers, two Arista 
   switches, one Junos device.

    IP info
    pynet-rtr1 (Cisco IOS)  184.105.247.70
    pynet-rtr2 (Cisco IOS)  184.105.247.71
    pynet-sw1  (Arista EOS) 184.105.247.72
    pynet-sw2  (Arista EOS) 184.105.247.73
    juniper-srx             184.105.247.76

2. Retrieve the 'model' from the returned data structure and print the model 
   to standard out.

3. Define the devices that you use in a Python file that you import.

4. [Optional] - Define the devices that you use in a YAML file that you read in.
